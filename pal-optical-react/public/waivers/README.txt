PLACE YOUR FORM-FILLABLE PDF FILES HERE

The POST app has buttons that open these PDFs in a new tab so patients can sign them directly on screen.

Required filenames:
  - waiver-expired-rx.pdf
  - waiver-patient-own-frame.pdf
  - waiver-frame-thickness.pdf
  - waiver-child-poly.pdf
  - waiver-lined-to-no-line.pdf
  - waiver-semi-rimless.pdf
  - waiver-remake.pdf

How to create them:
  1. Open the existing HTML waivers in /printable-docs/
  2. Use a tool like Adobe Acrobat, PDFescape, or DocHub to create form-fillable PDFs
  3. Save with the exact filenames listed above
  4. Place them in this folder
  5. The app will automatically link to them

No restart required — the files are served statically.
